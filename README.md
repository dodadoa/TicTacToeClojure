# TIC-TAC-TOE in clojure
-------------------------
** to play **

```
- clone this repo
- lein deps
- lein run
```

** how to play **

```
 px ix x-axis index
 py is y axis index

 px → 0     1     2
 py
 ↓  |-----|-----|-----|
 0  |     |     |     |
    |-----|-----|-----|
 1  |     |     |     |
    |-----|-----|-----|
 2  |     |     |     |
    |-----|-----|-----|

you must fill in only 'number number' format
otherwise it might be crash
```
